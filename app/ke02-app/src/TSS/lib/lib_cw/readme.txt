TSS Libraries for CodeWarrior Development Studio Special Edition Microcontrollers 
version 6.3 or higher and CodeWarrior for MCU v 10.1 or higher
=================================================================================

This folder contains platform-dependent TSS precompiled libraries 
for CodeWarrior Development Studio Special Edition Microcontrollers 
version 6.3 or higher and CodeWarrior for MCU v 10.1 or higher. Please see 
the Reference Manual (TSSAPIRM) for more information about how to use this library.
The suffix of the library file name informs about the MCU platform 
which this file is intended for. 

Library support files are located in <install_path>/lib/shared folder
(<install_path> is typically C:\Program Files\Freescale\Freescale TSS x.xx)

Following versions of libraries are currently provided for CodeWarrior 
Development Studio Special Edition Microcontrollers version 6.3 or higher and 
CodeWarrior for MCU v 10.1 or higher:
---------------------------------------------------------------------------------
TSS_S08.lib - Freescale’s HCS08 platform
TSS_CFV1.a - ColdFire®V1 and Colfire+ plaform
TSS_KXX_M4.a - ARM®Cortex™-M4 Kinetis platform
