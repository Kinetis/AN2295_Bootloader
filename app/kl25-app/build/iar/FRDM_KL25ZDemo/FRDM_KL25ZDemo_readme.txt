The platinum project supports the following boards and platforms:

Firebird daughter cards:
- X-L2K-DC-80LQFP

Tower cards:
- XTWR-K25LN128

BACES:
-ASB831

In order to switch to a different board:

1. Select the project for the overall platform (firebird, tower, or baces). 

2. Select the desired memory configuration from the drop menu in the project file. This determines the SRAM and/or flash configuration used.

NOTE: Do not remove the "IAR" or platform define ("FIREBIRD", "TOWER", or "BACES") from the defined symbols list.
